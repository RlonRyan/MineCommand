Command: tp
====================

About:
--------------------
Teleports the user to a location.

Usage:
--------------------
```
tp [-x:userpos] [-y:userpos] [-z:userpos] [-d:userpos] 
```

Parameters:
--------------------
```
 - x: The x-pos to teleport to
 - y: The y-pos to teleport to
 - z: The z-pos to teleport to
 - d: The dimension to teleport to

```
