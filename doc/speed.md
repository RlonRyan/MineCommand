Command: speed
====================

About:
--------------------
Changes the user's movement multiplier. Limited to range between 1 and 10.

Usage:
--------------------
```
speed [-p:player] [-s:integer] 
```

Parameters:
--------------------
```
 - p: The target player.
 - s: Speed mulitiplier

```
